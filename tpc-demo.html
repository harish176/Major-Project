ğŸ’¡ Usage Instructions

Frontend Usage:

  1. Navigate to Admin Dashboard in your React app
  2. Go to TPC section
  3. Click on "TPC Members" tab
  4. Use "Add Member" button to create new members
  5. Select appropriate category (faculty/b-tech/m-tech/mca/phd)
  6. Fill in category-specific fields automatically shown
  7. Save to create the member

Category System:

  • faculty - Shows Employee ID, Designation fields
  • b-tech/m-tech/mca/phd - Shows Roll Number, Year, Semester fields

Role Options:

  • Head, Co-Head, Member, Coordinator, Assistant Coordinator
